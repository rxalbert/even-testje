Quick answers to common questions.

Q1 What is this project about?
This project demonstrates a small example FAQ page with CSS-based styling for distinct question and answer sections.
Q2 How are questions and answers styled?
Questions use stronger weight and a small badge; answers use a softer color, increased line-height, and padding to improve readability.
Q3 Can I reuse this layout?
Yes — the CSS and HTML are simple to copy into other pages or adapt for dynamic content if needed.
